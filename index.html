About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, organise site records, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

I set up and maintained the project check infrastructure for SiteSpy. This includes project-check.js which scans the repository for common configuration issues, the GitHub Actions workflow (project-checks.yml) that runs these checks automatically on every push, and Testing_Checklist.md which documents the manual verification steps the team follows before a release. Consistent project checks caught integration problems early and kept the main branch stable throughout the semester.

My Reflection

Automating project checks taught me that a good CI pipeline is invisible when it works and invaluable when something breaks. I had to learn how GitHub Actions runner environments differ from local machines, which caused several confusing failures before I understood path and environment variable handling. The testing checklist made our release process repeatable.
