About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

I set up and maintained the project check infrastructure for SiteSpy. This includes project-check.js which scans the repository for common configuration issues, the GitHub Actions workflow (project-checks.yml) that runs these checks automatically on every push, and Testing_Checklist.md which documents the manual verification steps.
